# Phantom Tool

## 🎯 Описание устройства

**Phantom Tool** — это универсальный пентест-девайс на базе ESP32, разработанный командой **VLRX-Spy**. Устройство предназначено для перехвата радиосигналов, копирования NFC-меток и анализа уязвимостей систем безопасности.

### 🕵️‍♂️ Ключевые особенности

- **Невидимый перехват** — возможность перехвата и мониторинга радиоканала
- **Многофункциональность** — поддержка Wi-Fi, Bluetooth, NFC, RF протоколов
- **Доступность** — себестоимость ~2000–3500₽
- **Простота использования** — OLED дисплей для визуализации данных и управления устройством

### 🔧 Наши технологии

- ESP32 (Wi-Fi + Bluetooth) ESP32-DevKitC 
- NFC модули 
PN532
- RF модули 
CC1101
- OLED дисплеи для визуализации данных
Oled дисплей 0.96 I2C 128х64 (Белый) (1 шт)
